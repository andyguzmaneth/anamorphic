## Codebase Patterns
- Use `~/.cargo/bin/circom` (v2.2.3) for circuit compilation — the npm `circom` (v0.5.46) is the deprecated JS version
- Circom build output goes to `circuits/build/` (must create dir before compiling)
- Foundry project lives in `contracts/` with forge-std as a git submodule
- Rust workspace root is `relayer/Cargo.toml` with members: stealth-crypto (lib), relayer (bin)
- Root `package.json` has snarkjs and circomlib as devDependencies
- stealth-crypto uses k256 (secp256k1 ECDH/arithmetic), tiny-keccak (keccak256), rand, hex, aes-gcm, hkdf, sha2
- Instruction module: `Instruction` struct serializes to 141 bytes fixed-width big-endian; `encrypt_instruction` / `decrypt_instruction` use HKDF(shared_secret) -> AES-256-GCM
- Encryption format: nonce (12 bytes) || ciphertext || GCM tag — aes-gcm crate appends tag to ciphertext internally
- Stealth address protocol: ECDH shared secret -> keccak256(compressed point) -> reduce to scalar -> scalar*G + spend_pubkey
- `keccak256` and `pubkey_to_eth_address` are public in stealth_address module for reuse
- Solc 0.8.21 has an internal Natspec compiler bug — pin `solc_version = "0.8.24"` in foundry.toml
- StealthEscrow uses 7-day challenge period and 14-day recovery timeout (total 21 days before user recovery)
